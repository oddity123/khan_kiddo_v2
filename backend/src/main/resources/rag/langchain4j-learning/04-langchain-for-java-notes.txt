【文档类型：LangChain for Java 学习笔记】

LangChain for Java 即 LangChain4j，是 JVM 上构建 LLM 应用的 Java 库。

本次 Easy RAG 练习要点：
1. 依赖 langchain4j-easy-rag 做文档分块；嵌入模型使用通义千问 text-embedding-v3（中文检索更准确）
2. ClassPathDocumentLoader 从 rag/langchain4j-learning/ 加载 .txt 文档
3. EmbeddingStoreIngestor + minScore 过滤，避免 SQL 等工作碎片误命中
4. EmbeddingStoreContentRetriever 作为 ContentRetriever 接入 AiServices
5. 返回 TokenStream，经 SSE 推送到前端

与通用教程的区别：
本 Demo 的知识库不是 LangChain4j 官方文档，而是用户自己的「自我介绍」和「工作相关文件」。
提问应围绕这些私人文档内容，例如个人背景、项目职责、学习笔记。
