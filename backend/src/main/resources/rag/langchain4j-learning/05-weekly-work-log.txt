# 接口文档
## 睿策
查询内数表接口: internalDataTable/listQuery

查询内数表字段详细接口: internalDataTable/queryTable

引入配置-查询其它服务: /statistical/getServiceList

  
/template/findTreeBaseVarAndBaseArray 一般变量  
/template/decision/findTreeVarBaseAllArrayByType 添加数组对象  
/template/decision/findArrayObjectSubFieldV2 数组对象下的属性, 

+ 关联字段也是这个接口
+ configTypeEnum来判断是数组下的一般字段还是关联字段, INDICATOR: 一般, RELATION: 关联

## 外数
选择变量:

template/outsideStatic/findTreeBaseVarAndBaseArray: 选择一般变量 

template/findTreeVarBaseAllArrayByType: 选择数组对象

outsideServiceSaveDataConfig/findArrayObjectSubField: 选择数组对象下的变量

outsideServiceSaveDataConfig/findArrayObjectSubFieldV2: 选择数组对象关联字段的变量
