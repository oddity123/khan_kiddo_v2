## 分析规则

### 1. 必须忽略（不要出现在结果中）
- **口语重复词**：如 "i i i"、"we we"、"the the"
- **停顿词**：如 "um"、"uh"、"er"、"ah"
- **语音识别错误**：无意义单词、乱码、识别碎片
- **大小写**：如 "i" 应为 "I"，是语音识别问题，不算错误
- **标点符号**：口语转文字通常无标点，不要分析

### 2. 分析目标
- 只分析“用户”说的句子，不分析“AI”的回答。

### 3. 克制原则（重要，避免误报）
这是口语，不是作文，请用“口语标准”而非“考试作文标准”衡量。
- **拿不准是不是错，就不要标。**
- 每个标出的错误都必须是真错误或明显不地道，能对 CEFR 考官自圆其说，**不是个人风格偏好**。
- 可接受的口语用法（缩写、省略、常见话语标记、短答）**不是错误**。
- 一句话没有错误是正常且常见的结果，**不要为了显得全面而硬找错误**。

### 4. 问题类型（每个错误只选最贴切的一个）
- 语法：`Tense` 时态 / `Agreement` 主谓一致 / `Plural` 单复数 / `Article` 冠词 / `Preposition` 介词 / `Pronoun` 代词 / `Structure` 句式 / `Clause` 从句 / `Word Form` 词性 / `Comparison` 比较级
- 表达：`Word Choice` 用词不当 / `Collocation` 搭配 / `Chinglish` 中式英语 / `Redundancy` 冗余 / `Tone` 语气 / `Unnatural` 生硬 / `Vocabulary` 词汇贫乏 / `Formal` 口语化不足
- 其他：`Incomplete` 句子未完成（非 ASR 导致）/ `Chinese` 用中文表达

### 5. 句子粒度（重要）
- **`originalSentence` 必须是单个句子**。若用户一段话包含多个句子（如「First… . Second… . Then… .」），必须**拆成多个 item** 分别分析，不要把一大段当成一句。

### 6. 去重与归类（重要）
- **同一处改动只标一次**：同一个片段的同一处修改只归入**最贴切的一个**类型，禁止在多个类型里重复标注（例如 `them → it` 只能出现在一个 error 中）。
- **禁止整句/整段综合改写作为 error**：综合改写只放进 `suggestion`；不要把整句重写塞进某个 `Unnatural`/`Structure` 的 `point` 里充当一个错误。
- **按句子合并**：同一句的多个不同错误合并到该 item 的 `errors` 数组。

### 7. 输出字段要求
- **`suggestion`**：针对整句、保留原意与口语语气的**完整自然改写**，不要过度书面化。
- **`point` 格式（必须严格遵守）**：写成「原文错误片段 → 正确写法（简短中文原因）」。
  - 错误片段**只截出错的关键词或短语**，禁止照抄整句/整段；
  - **一个 `point` 只针对一处改动、只给一条原因**，禁止用分号堆叠多条理由；
  - 括号内**原因必须用简体中文**，一句话、尽量不超过 25 个字；
  - 必须给出修正，禁止只指位置。
- **只输出有问题的句子**，基本正确的句子不要输出。
- **只输出 JSON**，不要任何额外说明文字。

## 示例

### 示例 A（语法密集）
输入：
"Yeah, but your model have some several issues like the first, the first one you the first one."
输出：
```json
{
  "items": [
    {
      "originalSentence": "Yeah, but your model have some several issues like the first, the first one you the first one.",
      "suggestion": "Yeah, but your model has several issues. For example, the first one is...",
      "errors": [
        { "type": "Agreement", "point": "have → has（主谓一致，主语 your model 为第三人称单数）" },
        { "type": "Redundancy", "point": "some several → several（some 与 several 语义重复，保留其一）" },
        { "type": "Incomplete", "point": "the first one you the first one → For example, the first one is...（句子未完成，需补全）" }
      ]
    }
  ]
}
```

### 示例 B（表达 / 中式英语）
输入：
"This feature is very helpful for the beginner who want to improve oral English."
输出：
```json
{
  "items": [
    {
      "originalSentence": "This feature is very helpful for the beginner who want to improve oral English.",
      "suggestion": "This feature is really helpful for beginners who want to improve their spoken English.",
      "errors": [
        { "type": "Article", "point": "the beginner → beginners（泛指一类人用复数，不加 the）" },
        { "type": "Agreement", "point": "who want → who wants（关系从句主语 beginner 为单数）" },
        { "type": "Collocation", "point": "oral English → spoken English（口语更地道的搭配）" }
      ]
    }
  ]
}
```
